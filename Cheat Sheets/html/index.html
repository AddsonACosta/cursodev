CÓDIGOS DE HTML

HTML Cheat Sheet



HTML Básico

<!DOCTYPE> Define o tipo de documento como HTML.
<html> Define um documento html.
<head> Contém a informação metadata do documento.
<title> Define o título do documento.
<body> Define o corpo, parte visível do documento.
<h1> a <h6> Define os tipos e tamanhos de títulos.
<p> Define o parágrafo.
<br> Insere uma quebra de linha.
<hr> Define uma mudança temática no conteúdo.
<!--...--> Define um comentário.

Formatação

<abbr> Define uma abreviatura ou um acrônimo.
<address> Define uma informação de contato do autor do documento ou artigo.
<b> Define um texto em negrito.
<bdi> Isola uma parte do texto que pode estar formatada em uma direção diferente de outro texto fora dele.
<bdo> Substitui a direção do texto atual.
<blockquote> Define uma seção que é citada de outra fonte.
<cite> Define o título de uma obra.
<code> Define uma porção de código de computador.
<del> Define o texto que foi excluído de um documento.
<dfn> Especifica um termo que será definido no conteúdo.
<em> Define o texto enfatizado.
<i> Define um texto em itálico.
<ins> Define um texto que foi inserido em um documento.
<kbd> Define uma entrada do teclado.
<mark> Define o texto marcado/destacado.
<meter> Define uma medida escalar dentro de um intervalo conhecido (um medidor).
